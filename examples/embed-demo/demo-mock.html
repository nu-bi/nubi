Nubi Dashboard Embed — Demo

This page demonstrates the <nubi-dashboard> web component (M3-C). No live backend is required — the component always shows a table via its built-in sample-data fallback when the backend is unreachable. Open DevTools to see the nubi:ready / nubi:query-run events.
