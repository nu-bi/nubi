## Strength

Best-in-class collaborative Python notebook UX; strong AI-assisted analysis (Magic AI, Notebook Agent, Semantic Model agent); broad data source connectivity.

## Limitation

Per-session cloud kernel is the main cost driver — scales linearly with concurrent users. Embedding is expensive and bolt-on (Enterprise-only). SVG viz chokes past ~50 k rows. No self-host option.

## Notes

Hex cold-start latency is 10–30 s on free/team tiers. Compute billed per-minute: Large 16 GB $0.32/hr, XL 32 GB $0.65/hr, 2XL 64 GB $1.29/hr. Medium compute included on all paid plans.
